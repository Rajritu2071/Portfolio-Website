## Overview

Personal portfolio website built to showcase my projects, skills, and experience in AI/ML engineering and CUDA systems. Designed to be fast, clean, and mobile-responsive.

## Features

- **Projects showcase** — CUDA Optimisation, AI Smart Grid, InterviewAI
- **Skills section** — Tech stack and tools
- **About me** — Background and interests
- **Contact** — Links to LinkedIn, X, and GitHub
- **Mobile responsive** — Works across all screen sizes

- **HTML5 / CSS3** — Structure and styling
- **Vanilla JavaScript** — Interactivity
- **Netlify** — Hosting and continuous deployment
